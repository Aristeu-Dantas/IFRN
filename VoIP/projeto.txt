#PROJETO AGI

1 – Os projetos utilizando AGI (Asterisk Gateway Interface) tem os seguintes requisitos / orientações gerais: 
    - Pode ser realizado de forma individual ou em duplas.
    - A aplicação AGI pode ser escrita em qualquer linguagem de programação.
    - A proposta do projeto (autores; breve descrição; linguagem de programação) a ser desenvolvido (por cada grupo) deve ser enviada pelo classroom (ou discutida em sala de aula). Elas serão analisadas e aprovadas / reprovadas (com ou sem sugestões).

2 – Os requisitos para a aplicação AGI a ser desenvolvida são os seguintes:
    - Deve haver o atendimento por meio de menu de voz, descrevendo brevemente a finalidade da aplicação e solicitando a digitação dos dados necessários (CPF, matrícula, usuários/senhas, códigos etc.).
    - Os dados digitados no ramal devem ser utilizados para interagir com, pelo menos, um ente externo ao Asterisk como banco de dados, Web service e/ou outro tipo de API, servidores de outros serviços e/ou arquivos locais etc.
    - Antes de finalizar, a aplicação deve gerar (pelo menos) uma resposta audível confirmando ou não o sucesso da operação. Obviamente, também pode haver outros tipos de resposta (por exemplo em uma página web).

3 – Uma apresentação e demonstração da aplicação em funcionamento deverá ser realizada (pelos grupos), devendo durar em torno de 15 minutos.  

Entrega das propostas até 26/07/2024 (via classroom / sala de aula).
Entrega dos projetos até o dia 16/09/2024 e apresentações nos dias 16/09/2024 e 18/09/2024.

Devem ser entregues a AGI e todos os outros arquivos necessários para seu funcionamento (áudios, extensions etc).

Descrição:
Este projeto visa desenvolver um sistema de enquetes automatizado, acessível por telefone, que permite coletar dados de forma eficiente e rápida. Ao ligar para o sistema, o usuário será autenticado utilizando seus dados cadastrais (exemplo: CPF, matrícula ou outro identificador único). Após a autenticação, o sistema apresentará uma série de perguntas pré-definidas, relacionadas a diversos temas. Ao final as respostas serão gravadas em um arquivo e guardadas num banco de dados.

Linguagem de programação: Python
